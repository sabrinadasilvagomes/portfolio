Projecten

User Centred Design

Focus: UX - UI

Voor dit project moest ik voor "Maas International” een UX-ontwerp verbetering onderzoeken, waardoor de gebruikerservaring met de koffieautomaten werd verbeterd. Deze verbetering moest tot stand komen door het doen van verschillenden soorten onderzoek naar de doelgroep.

Het proces

Onderzoek

Voor User Centred Design was het van belang dat gebruiksvriendelijkheid (zoals de naam van het vak ook al suggereert) centraal stond. Door allerlei onderzoeken te doen, kon ik een passend advies formuleren. Voorbeelden van uitgevoerde onderzoeken zijn: Persona, Storyboards, Triangulatie check en interviews.

Waar liep ik tegen aan?

Een lastig punt was dat je alle onderzoeksgegevens weer aan elkaar moest kunnen koppelen. Hier liep ik soms een beetje vast. Ik wist namelijk niet helemaal goed hoe ik dit het beste kon doen. Een verbeterpunt voor mijzelf is het noteren van ideeën. Door het in mijn hoofd te bedenken wordt het toch vaak een warboel van informatie.

Ten tijde van dit project konden wij helaas ook geen les op locatie volgen vanwege Covid-19. Ik heb uiteindelijk maar één moment gehad waar ik echt fieldresearch kon doen. Dit was ook best lastig voor dit project. Hierdoor was dit onderzoek misschien niet zo nauwkeurig als ik had gewild.

Hoe heb ik dit opgelost?

Door alle onderzoeksgegevens op een rijtje te zetten, werd het pas echt duidelijk dat dit ook echt aan elkaar te koppelen was. Met deze conclusie kon ik dus ook echt een advies formuleren. Met het Covid-19 probleem was het iets lastiger. Ik heb zoveel mogelijk informatie van internet proberen te halen, dus de deskresearch. Op het moment dat het wel veilig was om naar locatie te gaan, heb ik alle mogelijke scenario's proberen af te spelen, zodat ik mijzelf kon inleven in de gebruiker.

Wat neem ik mee naar een volgend project?

Tijdens een volgend project zal ik ook hier weer sneller dingen opschrijven. (ik heb ontzettend veel notitieboekjes, waarom gebruik ik ze dan niet?!) Verder zal ik ook proberen om toch op meer manieren onderzoek te doen. Denk bijvoorbeeld aan momenten dat sommige onderzoeken afvallen door omstandigheden. Het is dan van belang dat je dan wel andere methodes kan toepassen.

Wat vond ik van dit project?

User Centred Design was één van mijn lievelingsvakken. Het was erg interessant om te zien welke onderzoeksmethodes er allemaal zijn en hoe je deze het beste kon uitvoeren. Het was erg leuk om te zien dat je alles uiteindelijk ook weer aan elkaar kon koppelen (als het onderzoek goed was uitgevoerd). Ik heb ook het idee gehad dat ik hier ook al mijn creativiteit in kwijt kon als het gaat om het ontwerpen van een oplossing. Denk hierbij aan het maken van storyboards en het vormgeven van onderzoek.
